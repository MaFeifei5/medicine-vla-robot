# Project Rules

## Goal
Build a graduation project for a VLA-based medicine picking robot.

## Hardware
- UR robot arm
- Intel RealSense L515
- Zhiyuan gripper

## Rules
- Prefer minimal changes
- Do not refactor unrelated files
- Always explain which files will be changed before editing
- Always provide runnable test commands
- Keep the main branch runnable

## Current Focus
1. Robot-camera baseline
2. Data recording
3. LeRobot training pipeline
